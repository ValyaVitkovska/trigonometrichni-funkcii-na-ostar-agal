Отношения \(\alpha\)

Основни дефиниции

В правоъгълен триъгълник \(\triangle ABC\) (с прав ъгъл при върха \(C\)), функциите се дефинират като отношения на страните. Нека разгледаме острия ъгъл \(\alpha\):

Синус

\(\sin \alpha = \frac{a}{c}\)

Срещулежащ / Хипотенуза

Косинус

\(\cos \alpha = \frac{b}{c}\)

Прилежащ / Хипотенуза

Тангенс (tg)

\(\operatorname{tg} \alpha = \frac{a}{b}\)

Срещулежащ / Прилежащ

Котангенс (cotg)

\(\operatorname{cotg} \alpha = \frac{b}{a}\)

Прилежащ / Срещулежащ

Основни тъждества

  • \(\sin^2 \alpha + \cos^2 \alpha = 1\)
  • \(\operatorname{tg} \alpha = \frac{\sin \alpha}{\cos \alpha} \quad \text{и} \quad \operatorname{cotg} \alpha = \frac{\cos \alpha}{\sin \alpha}\)
  • \(\operatorname{tg} \alpha \cdot \operatorname{cotg} \alpha = 1\)

Връзка между \(\alpha\) и \(\beta\)

Тъй като \(\alpha + \beta = 90^\circ\), функциите преминават в своите ко-функции:

  • \(\sin \alpha = \cos(90^\circ - \alpha) = \cos \beta\)
  • \(\cos \alpha = \sin(90^\circ - \alpha) = \sin \beta\)
  • \(\operatorname{tg} \alpha = \operatorname{cotg}(90^\circ - \alpha) = \operatorname{cotg} \beta\)
  • \(\operatorname{cotg} \alpha = \operatorname{tg}(90^\circ - \alpha) = \operatorname{tg} \beta\)
